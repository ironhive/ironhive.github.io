Shaolin-Do Trainer

Personal project - training randomizer

What is this thing?

In Shaolin-Do kung fu, of which I am a student, we learn a fairly large set of movements and most basic of them are numbered 1-20 for one set and 1-30 for another. It easily becomes a habit to practice these sequentially, since that is the way they are initially taught. I wanted a training companion to help with improvisation and break my dependence on recalling movements in order.

I defined the following requirements for this app:

  • Ability to select the type(s) of movements I want to practice.
  • This is related to fighting and self-defense, so urgency is a factor. I need a timer of some sort.
  • The timer should have speed settings so I can increase it as my reaction time improves.
  • Learning which movements work well together is critical, so I need the option to have combinations of movements randomly put together.
  • Run on my desktop, tablet or phone.
  • Have an option for the commands to be spoken aloud so I don't have to look at the screen.
