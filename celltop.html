Feature Phone Customizable UI

My Role

At the peak of Blackberry prominence, before Apple had released the iPhone, I helped design and develop a data-driven, customizable UI for feature phones. Most of the development was using Qualcomm's acquired UI One tech, written in the proprietary TrigML.

The UI won CTIA's Best in Show award in 2007
